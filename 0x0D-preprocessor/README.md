# 0x0D-preprocessor
## 0. Object-like Macro
Create a header file that defines a macro named `SIZE` as an abbreviation for the token 1024.
### Related files
`0-object_like_macro.h` `0-main.c`
### Run this command
```bash
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 0-main.c -o a && ./a
```
